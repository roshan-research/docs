    این بخش از ای‌پی‌آی برای ترجمه متن از یک زبان به زبان دیگر استفاده می‌شود. کاربر می‌تواند متن موردنظر را همراه با زبان مقصد ارسال کند و ترجمه آن را دریافت نماید.

    values = { "text": "Hello, how are you? I want to know what day it is today and what time it is.", "lan": "fa" }

    متن (text): در این قسمت، متن اصلی که نیاز به ترجمه دارد وارد می‌شود.

    زبان مقصد (lan): مشخص می‌کند که متن به چه زبانی ترجمه شود. مقادیر قابل قبول: en : انگلیسی fa : فارسی ar :عربی

    POST /translate/

    این بخش از ای‌پی‌آی برای خلاصه‌سازی متون طولانی مورد استفاده قرار می‌گیرد. کاربران می‌توانند متن خود را همراه با نوع خلاصه‌سازی و سطح دشواری ارسال کنند و خلاصه‌ای متناسب با درخواست خود دریافت نمایند.

    text : متن متن اصلی که قرار است خلاصه شود.

    type : نوع ترجمه مشخص می‌کند که خروجی به چه فرمتی باشد: Word : خلاصه به‌صورت متن پیوسته و کامل. Conversation : خلاصه به‌صورت مکالمه‌ای. Bullet : خلاصه به‌صورت نکات کلیدی و فهرست‌وار.

    سطح خلاصه‌سازی (level): میزان حجم محتوای خلاصه شده را تعیین می‌کند: Level_1 : کمترین میزان خلاصه‌سازی Level_2 : خلاصه‌ای متعادل Level_3 : بیشترین میزان خلاصه‌سازی

    POST /summarize/
