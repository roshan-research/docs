    فهم

    این ای‌پی‌آی برای انجام عملیات ترجمه، خلاصه‌سازی و گفت‌وگو طراحی شده است.

    برای استفاده از این ای‌پی‌آی نیازمند توکن هستید. این توکن را از بخش فروش دریافت کنید.

    ترجمه

    یک متن را از انگلیسی به فارسی یا از فارسی به انگلیسی ترجمه می‌کند. زبان‌های دیگر پشتیبانی نمی‌شود.

    ترجمه متن انگلیسی به فارسی

    Request

    {
    "text": "Hello. How are you?",
    "to": "fa"
}

    curl  --request POST \ 
      --header "Content-Type: application/json" --header "Authorization: Token YOUR_API_TOKEN" \
      --data-binary '{
    "text": "Hello. How are you?",
    "to": "fa"
}' \
      https://fahm.roshan-ai.ir/api/translation/

    import requests

values = {
    "text": "Hello. How are you?",
    "to": "fa"
}

headers = {
  'Authorization': 'Token YOUR_API_TOKEN',
}
response = requests.post('https://fahm.roshan-ai.ir/api/translation/', data=values, headers=headers)

print(response.json())

    Response

        "result": "```\nسلام. چطوری؟\n```"

    خلاصه‌سازی
