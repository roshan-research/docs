    فهم

    این ای‌پی‌آی برای انجام عملیات ترجمه، خلاصه‌سازی و گفت‌وگو طراحی شده است.

    ترجمه

    ترجمه متن

    Request

    {
    "text": "Hello, how are you? I want to know what day it is today and what time it is.",
    "lan": "fa"
}

    curl  --request POST \ 
      --header "Content-Type: application/json" --header "Authorization: Token `YOUR_API_TOKEN`" \
      --data-binary '{
    "text": "Hello, how are you? I want to know what day it is today and what time it is.",
    "lan": "fa"
}' \
      https://fahm.roshan-ai.ir/api/translate/

    import requests

values = {
    "text": "Hello, how are you? I want to know what day it is today and what time it is.",
    "lan": "fa"
}

headers = {
  'Authorization': 'Token `YOUR_API_TOKEN`',
}
response = requests.post('https://fahm.roshan-ai.ir/api/translate/', data=values, headers=headers)

print(response.json())

    Response 

    {
    "translated_text": "سلام، حال شما چطور است؟ می‌خواهم بدانم امروز چه روزی است و ساعت چند است."
}

    این بخش از ای‌پی‌آی برای ترجمه متن از یک زبان به زبان دیگر استفاده می‌شود. کاربر می‌تواند متن موردنظر را همراه با زبان مقصد ارسال کند و ترجمه آن را دریافت نماید.

    POST /translate/

    خلاصه‌سازی

    خلاصه‌سازی متن

    Request

    "text=\"This short story is set in Madagascar and France. It tells about wine and love - and hate. And about a winepress, the machine used to press the grapes from which wine is made\"\ntype=\"word\"\nlevel=\"level_1\"\n"

    curl  --request POST \ 
      --header "Content-Type: multipart/form-data" --header "Authorization: Token `YOUR_API_TOKEN`" \
      --data-binary '"text=\"This short story is set in Madagascar and France. It tells about wine and love - and hate. And about a winepress, the machine used to press the grapes from which wine is made\"\ntype=\"word\"\nlevel=\"level_1\"\n"' \
      https://fahm.roshan-ai.ir/api/summarize/

    import requests

values = "text=\"This short story is set in Madagascar and France. It tells about wine and love - and hate. And about a winepress, the machine used to press the grapes from which wine is made\"\ntype=\"word\"\nlevel=\"level_1\"\n"

headers = {
  'Content-Type': 'multipart/form-data','Authorization': 'Token `YOUR_API_TOKEN`',
}
response = requests.post('https://fahm.roshan-ai.ir/api/summarize/', data=values, headers=headers)

print(response.json())

    Response 

    {
    "summary": "A short story about wine, love, hate, and a winepress set in Madagascar and France."
}

    این بخش از ای‌پی‌آی برای خلاصه‌سازی متون طولانی مورد استفاده قرار می‌گیرد. کاربران می‌توانند متن خود را همراه با نوع خلاصه‌سازی و سطح دشواری ارسال کنند و خلاصه‌ای متناسب با درخواست خود دریافت نمایند.

    POST /summarize/

    import requests

values = {
    "query": "کدام شرکت ها از سیستم پرگار استفاده میکنند",
    "username": "YOUR_USERNAME",
    "token": "YOUR_API_TOKEN"

  'Authorization': 'Token `YOUR_API_TOKEN`',
}
response = requests.post('https://fahm.roshan-ai.ir/api/chat/stream/', data=values, headers=headers)

    Response 

    {
    "response": "شرکت‌های A، B و C از سیستم پرگار استفاده می‌کنند."

    این بخش از ای‌پی‌آی امکان گفت‌وگو با هوش مصنوعی را فراهم می‌کند. کاربران می‌توانند سوالات خود را ارسال کرده و پاسخ متنی دریافت کنند. این قابلیت می‌تواند برای پاسخ به سوالات عمومی، ارائه اطلاعات و تعاملات هوشمندانه مورد استفاده قرار گیرد.

    POST /chat/stream/
