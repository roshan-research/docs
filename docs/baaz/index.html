    باز

    باز یک ای‌پی‌آی بومی برای جستجو در محتوای فارسی است. برای دسترسی به این ای‌پی‌آی به یک TOKEN_KEY نیاز دارید که می‌توانید با ارسال یک ایمیل به baaz@roshan-ai.ir درخواست دهید.

    ایندکس‌کردن و ویرایش محتوا

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name

    این اندپوینت برای افزودن، جایگزینی و ویرایش اسناد در یک ایندکس مشخص به کار می‌رود. قبل از جستجو ابتدا باید داده‌ها را ایندکس کنید.

    مثال: ویرایش محتوای موجود

    {
    "items": [
        {
            "type": "movie",
            "id": 1,
            "importance": 200
        }
    ]
}

    curl  --request PUT \ 
      --header "Content-Type: application/json" --header "Authorization: Token TOKEN_KEY" \
      --data-binary '{
    "items": [
        {
            "type": "movie",
            "id": 1,
            "importance": 200
        }
    ]
}' \
      https://baaz.roshan-ai.ir/{index_name}/index

    import requests

values = {
    "items": [
        {
            "type": "movie",
            "id": 1,
            "importance": 200
        }
    ]
}

headers = {
  'Authorization': 'Token TOKEN_KEY',
}
response = requests.put('https://baaz.roshan-ai.ir/{index_name}/index', data=values, headers=headers)

print(response.json())

    Response 

    {
    "ids": [1]
}

    برای ویرایش بخشی از محتوای یک ایندکس، درخواست خود را با متد PUT ارسال کنید. در هر آیتم، واردکردن type و id برای یافتن محتوای مورد نظر ضروری است.

    PUT /{index_name}/index

    حذف ایندکس یا نوع

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name

    از این اندپوینت برای حذف کامل یک ایندکس یا حذف تمام محتواهای یک نوع خاص (type) استفاده می‌شود. رفتار این اندپوینت به بدنه درخواست بستگی دارد.

    حذف

    Request

    {
    "type": "type2"
}

    curl  --request DELETE \ 
      --header "Content-Type: application/json" --header "Authorization: Token TOKEN_KEY" \
      --data-binary '{
    "type": "type2"
}' \
      https://baaz.roshan-ai.ir/{index_name}/delete_index

    import requests

values = {
    "type": "type2"
}

headers = {
  'Authorization': 'Token TOKEN_KEY',
}
response = requests.delete('https://baaz.roshan-ai.ir/{index_name}/delete_index', data=values, headers=headers)

print(response.json())

    Response

  "index": "TESTDELETE",

  "weaviate": {
    "status": "success",
    "message": "Successfully deleted 0 documents of type 'type2'",

  },

    type(required)

    نوع اسنادی که باید حذف شوند.



    دریافت آمار ایندکس

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name

    مثال

      --data-binary '""' \
      https://baaz.roshan-ai.ir/{index_name}/stats

values = ""

  'Authorization': 'Token TOKEN_KEY',
}

print(response.json())

    Response 

    {
  "document": 3,
  "نامه": 1
}
    GET /{index_name}/stats

    مشاهدهٔ محتوا

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name
    نوع مورد نظر برای مشاهده. types
    شناسهٔ محتوا. id

    این اندپوینت محتوای کامل یک سند ایندکس‌شده را با استفاده از شناسه (id) آن بازیابی می‌کند.

    شناسهٔ ایندکس. index_name

    یافتن اسناد مشابه

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name

    با دریافت شناسهٔ یک محتوای ایندکس‌شده، شبیه‌ترین محتواها را به آن پیدا کرده و برمی‌گرداند.

    مثال

    Request

    {
    "document_id": "f2646930-64d7-47cc-93c1-e557777f1c489ad",
    "page_size": 10,
    "page": 1
}

    curl  --request POST \ 
      --header "Content-Type: application/json" --header "Authorization: Token TOKEN_KEY" \
      --data-binary '{
    "document_id": "f2646930-64d7-47cc-93c1-e557777f1c489ad",
    "page_size": 10,
    "page": 1
}' \
      https://baaz.roshan-ai.ir/{index_name}/similar/documents

    import requests

values = {

  'Authorization': 'Token TOKEN_KEY',

    بررسی وضعیت اسناد

    پارامترهای ورودی

    توضیحات کلید
    شناسهٔ ایندکس. index_name

        "https://baaz.roshan-ai.ir/pargar-6b8e.../نامه/view?id=c81d..."
    ]
}

    curl  --request POST \ 
      --header "Content-Type: application/json" --header "Authorization: Token TOKEN_KEY" \
      --data-binary '{
    "document_urls": [
        "https://baaz.roshan-ai.ir/pargar-6b8e.../نامه/view?id=c81d..."
    ]
}' \
      https://baaz.roshan-ai.ir/{index_name}/document_status

    import requests

values = {
    "document_urls": [
        "https://baaz.roshan-ai.ir/pargar-6b8e.../نامه/view?id=c81d..."
    ]
}

headers = {
  'Authorization': 'Token TOKEN_KEY',
}
response = requests.post('https://baaz.roshan-ai.ir/{index_name}/document_status', data=values, headers=headers)

print(response.json())

    Response 

    {
  "documents": {
    "https://baaz.roshan-ai.ir/pargar-6b8e.../نامه/view?id=c81d...": "true"
  }
}
    POST /{index_name}/document_status
    document_urls(required)

    Value: [undefined, ]
